多平台应用图标生成工具。

Icon Gen 用于把一张高质量图标快速生成 iOS、Android、macOS、Windows、watchOS 所需的多尺寸资源和标准目录结构。它是 lys Open Source 体系中的项目资产生成工具。

项目能力

覆盖应用发布前最常见的图标资产准备工作，重点是本地处理、结构化输出和跨平台规格统一。

多平台规格

支持 iOS、iPadOS、macOS、watchOS、Android 和 Windows 常用图标尺寸。

标准目录结构

可生成 Xcode 需要的 AppIcon.appiconset 和 Android 资源目录结构。

浏览器本地处理

Web 版使用 Canvas 在浏览器内处理图片，不上传原图，适合快速生成和临时处理。

使用入口

当前项目包含 Web 版和 Electron 桌面版。线上主入口是项目站，在线工具作为体验入口保留。
